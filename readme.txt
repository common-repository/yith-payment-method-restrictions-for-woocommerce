YITH Payment Method Restrictions for WooCommerce plugin allow to hide specific payment methods based on the products in cart.

== Description ==

YITH Payment Method Restrictions for WooCommerce gives you the possibility to hide specific payment methods based on the products in cart.


= Features: =

Create one or more payment restrictions based on the payment method selected.

= PREMIUM FEATURES of YITH Payment Method Restrictions for WooCommerce: =

* Create restriction rules for all the payment gateways enabled
* Set one or more conditions for each restriction rule
* Disable the payment gateway when one or more conditions occur
* Automatically switch the bank account to be used depending on the user's purchase conditions
* Notify a missing payment gateway with a custom message at checkout

== Installation ==

1. Unzip the downloaded zip file.
2. Upload the plugin folder into the `wp-content/plugins/` directory of your WordPress site.
3. Activate `YITH Payment Method Restrictions for WooCommerce` from Plugins page.

YITH Payment Method Restrictions for WooCommerce will add a new submenu called "Payment Method Restrictions" under "YITH Plugins" menu. You will able to configure all the plugin settings there.
